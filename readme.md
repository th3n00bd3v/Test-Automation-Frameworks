## Automation Frameworks

A repository for tracking practice automation scripts in JavaScript and Python.

The folder structure is as follows:
```
Automation Scripts
├── JavaScript
│   ├── Cypress
│   ├── Nightwatch JS
│   ├── Playwright
│   ├── Puppeteer
│   └── WebdriverIO
└── Python
    ├── Robot Framework
    └── Selenium
```
